Neuedist

Screenprinting is wizardry.

  • ← See all work

I design so many things for screens and thought it’d be fun to try something printed. Enter Neuedist, a product company that sells custom t-shirts and prints. Yes, the name is a typography pun. I’m ok with that.

Building Neuedist

I built Neuedist to be lean. I looked into a few options that could power an e-commerce site: Shopify, Gumroad, WordPress, Square Market. There were other services out there which would let me sell my wares but they didn’t quite fit with what I wanted.

I wanted to run this as cheaply as I could, so paying each month was out. I decided on Gumroad since I paid per transaction. I could run the site off of my existing host. Gumroad also let me manage coupon codes and inventory.

Using static HTML and Bootstrap, I was able to pull this together in short order to begin selling shirts. The site loaded so quickly it felt like magic. I’m hosting with Digital Ocean, which uses SSDs to serve their files. Man, they’re fast. I also learned about using things like gzip to optimize the page load (that was tons of fun, seriously).

Designing

I worked through several iterations of the logo, starting with a few sketches, hoping to blend something bold with something human. I ended up choosing one that had used the lovely Futura with a more human-feeling script for the tagline.

I didn’t end up with a photo-heavy background, but had fun exploring some various options. After a recent trip to New York, I applied a photo from the trip for a different feeling. I ultimately wanted something that would help the site run a little leaner and let the content shine, so I dumped it and went with a solid color. Turned out to be a good idea.

Launch Neuedist

What I learned

  • Setting up my own server was hard, fun.
  • Optimizing page speed can become an obsession.
  • There are ways to run an e-commerce site on next-to-no-budget.
  • Designing for screens is fun, funding your own screenprinting is just as great.
